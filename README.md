# Nigerian-Fashion-Company-Financial-Model
5-Year Financial Model and DCF Valuation for a Nigerian Fashion Company

## Overview
Developed a 5-year business forecasting model for a Nigerian fashion company to evaluate revenue growth, operating costs, profitability, cash flow, and funding requirements.

The model was designed to support strategic decision-making by analyzing how different business assumptions affect future performance.

## Key Features

- Revenue forecasting
- Cost and profitability analysis
- Three-statement financial model
- Working capital analysis
- Scenario analysis
- DCF valuation

## Skills Demonstrated

- Financial Analysis
- Business Analysis
- Forecasting
- Scenario Planning
- Microsoft Excel
